# Customer Data Analysis

## Project Overview

This project focuses on analyzing customer purchasing behavior using Python and data analytics libraries. The dataset contains customer demographics, spending patterns, membership details, and transaction history.

The objective of this project is to perform:

* Data Cleaning
* Data Preprocessing
* Exploratory Data Analysis (EDA)
* Customer Insights Generation

---

## Tools & Libraries Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## Dataset Features

The dataset includes:

* Customer ID
* Age
* Gender
* City Tier
* Membership Type
* Total Spend
* Number of Transactions
* Average Transaction Value
* Days Since Last Purchase
* Number of Visits
* Product Categories Purchased
* Discount Used
* High Value Customer Status

---

## Tasks Performed

### 1. Data Exploration

* Loaded dataset using Pandas
* Checked dataset shape and structure
* Generated statistical summary

### 2. Missing Value Handling

* Identified missing values
* Filled numerical missing values using the median
* Filled categorical missing values using mode

### 3. Duplicate Handling

* Identified duplicate customer records
* Removed duplicate customer IDs

### 4. Exploratory Data Analysis

* Analyzed customer demographics
* Examined membership distribution
* Identified high-value customers
* Created visualizations for better insights

---

## Key Insights

* Silver membership customers are the highest in number.
* Female customers slightly outnumber male customers.
* A significant portion of customers is not classified as high-value customers.
* Customer spending patterns vary significantly across membership types.

---

## Project Outcome

This analysis helps understand customer behavior, spending habits, and membership trends, which can support business decision-making and customer segmentation strategies.
